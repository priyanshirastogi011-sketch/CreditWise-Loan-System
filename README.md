# 💳 CreditWise Loan System

An end-to-end Machine Learning project that predicts whether a loan application will be approved using classification algorithms.

---

## 📌 Project Overview

CreditWise Loan System is a supervised machine learning project built to predict loan approval based on applicant details.

The project includes:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
